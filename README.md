# Users
users service provide authentication and users data storing.

You need `.env` file in the project root containing the following datas:

```.env
DB_USER=your_admin
DB_PASSWORD=a_password
DB_SSLMODE=disable

DB_PORT_DEV=60210
DB_HOST_DEV=localhost

DB_PORT_PROD=60210
DB_HOST_PROD=localhost
```
